# The Clueless Cook

Weet jij ook niet wat je moet eten vanavond? of wil je eens weten hoeveel calorieën er nou eigenlijk in een product zitten? Met de de Clueless Cook kan je deze informatie vinden of een lekker recept opzoeken.

## installatie:
Dit project is gemaakt met het React Framework, hiervoor heb je een IDE nodig (bijv. Webstorm). Daarnaast is het belangrijk om de benodigde dependencies en axios te installeren.

Voer in de terminal het commando `npm install` uit om de dependencies te installeren.

Voer in de terminal het commando `npm install axios` uit om axios te installeren

Nadat deze zijn geïnstalleerd kunnen de API key en ID worden ingevuld in het bestand `.env.dist` dit zorgt ervoor dat de benodigde informatie kan worden opgevraagd uit de Edamam API.

Hierna kan in de terminal het commando `npm start` worden uitgevoerd, hiermee wordt de applicatie opgestart in de developer modus. Deze opent automatisch als http://localhost:3000
